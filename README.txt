# Expense Tracker Application

## Overview

This is a React-based Expense Tracker UI with an Express REST API backend. Users can view, filter, and manage expense data through a simple interface.

---

## Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/) (version 14 or higher recommended)
- npm (comes bundled with Node.js)

---

## Installation Instructions

#Step 1: Extract the files to a folder on your computer

#Step 2: Open the command prompt or a terminal and run the commands in the folder you extracted the files into:

For the frontend (client folder)
	cd client
	npm install

For the backend (server folder) 
	cd ../server 
	npm install

#Step 3: Once it is finished installing, run this command to start the application. It must be run in the "server" folder.
	npm run dev
